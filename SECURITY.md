# Security Policy

## Supported Versions

Currently, only the latest commit on the `main` branch is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Main    | :white_check_mark: |
| Forks   | :x:                |

## Reporting a Vulnerability

Because this application runs entirely client-side without a backend database or server, traditional server-side vulnerabilities are mitigated. However, if you discover a security issue (such as an XSS vulnerability in how user inputs are handled by the charting library), please do not report it via public GitHub Issues.

Instead, please report it privately via email:
**pradhyumnar26@gmail.com**

You should receive a response within 48 hours. If the issue is verified, we will work to patch it and release an update as quickly as possible.
